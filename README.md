
# Tempo Color EDF 💙🤍❤️

A small unofficial package to get the color link to a day for Tempo EDF subscription 



## Installation

Install tempoedf with npm:

```bash
  npm i tempoedf
```
    
## Exemple

```javascript
tempo = require('tempoedf');
tempo.getTempoColor().then((color) => {
    console.log(color);
});
```
